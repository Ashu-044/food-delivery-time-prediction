# 🚀 Food Delivery Time Prediction System

> 🔍 Built during a Hackathon to solve real-world delivery time prediction challenges using Machine Learning.

A machine learning solution to accurately predict food delivery time using real-world data and regression techniques.

---

## 📌 Problem Statement

Food delivery platforms often rely on static rules (e.g., distance or average delivery time) to estimate delivery duration. These approaches fail to capture real-world variability such as delivery partner efficiency, order type, and geographic conditions.

This results in:

- Inaccurate delivery estimates  
- Reduced customer satisfaction  
- Inefficient logistics planning  

---

## 💼 Business Impact / Why This Matters

Accurate delivery time prediction helps:

- 📈 Improve customer trust and experience  
- 🚚 Optimize delivery operations  
- ⏱️ Reduce delays and cancellations  
- 📊 Enable data-driven decision making  

---

## 💡 Solution Overview

This project builds an end-to-end Machine Learning regression pipeline that:

- Cleans and preprocesses delivery data  
- Performs exploratory data analysis (EDA)  
- Engineers meaningful features  
- Trains and compares multiple models  
- Deploys the best model for real-time predictions  

---

## 📊 Dataset

The dataset contains:

- Delivery Partner Age  
- Delivery Partner Rating  
- Restaurant Latitude & Longitude  
- Delivery Location Latitude & Longitude  
- Order Type  
- Vehicle Type  
- Delivery Time (Target Variable)
